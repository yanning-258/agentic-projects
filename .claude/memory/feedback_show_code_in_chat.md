Show code in chat for the user to copy-paste. Do not write files directly unless explicitly asked.

**Why:** User prefers to paste code themselves — keeps them in control and helps learning retention.

**How to apply:** Default to showing code blocks in chat. Only use Write/Edit tools if the user explicitly says "write it for me" or "go ahead and create it".
